# 🎯 Система отслеживания и управления цветными объектами на базе ROS для TurtleBot3 🤖



## 📋 Описание проекта

Данный проект реализует полноценную систему автономного мобильного робота на базе ROS, способную в реальном времени обнаруживать, отслеживать и следовать за цветными объектами с помощью RGB-D камеры 🎨

Система использует цветовое пространство HSV для сегментации цели, объединяет данные о глубине для оценки её трёхмерного положения и обеспечивает надёжное поведение слежения с помощью конечного автомата 🎯



## ✨ Возможности

| Функция | Описание |
|:---|:---|
| 🎨 **Обнаружение цветного объекта** | Надёжная сегментация цели на основе цветового пространства HSV |
| 📍 **3D-локализация** | Использование данных глубины RGB-D для вычисления положения цели в системе координат камеры и робота |
| 🔄 **Управление конечным автоматом** | Стратегия отслеживания в четырех состояниях: `SEARCHING` → `ALIGNING` → `APPROACHING` → `REACHED` |
| 🏃 **Слежение за движущейся целью** | Сопровождение движущихся объектов в реальном времени |
| 🌈 **Поддержка нескольких цветов** | Зелёный 💚 / Оранжевый 🧡 / Красный ❤️ / Синий 💙 (выбор через ползунки OpenCV) |
| 📡 **Резервирование по лидару** (бета-версия) | Альтернативное измерение расстояния с помощью лидара при недействительных данных глубины |

## ⚙️ Установка и сборка

```bash
# Перейти в рабочее пространство
cd ~/catkin_ws/src

# Клонировать репозиторий
git clone https://github.com/Tywh202/RGB-D-Based-Target-Tracking-and-Control-for-Mobile-Robots.git

# Собрать
cd ~/catkin_ws
catkin_make

# Настроить окружение
source devel/setup.bash
```



## 🚀 Быстрый старт

### 1️⃣ Запуск среды симуляции

```bash
export TURTLEBOT3_MODEL=waffle
roslaunch image_pkg turtlebot3_balls.launch
```

### 2️⃣ Запуск отслеживания цели

**Основная версия** ⭐ (отчёт основан на этой версии):
```bash
rosrun image_pkg hsv_node_release.py
```

**Бета-версия** 🧪 (с резервированием по лидару):
```bash
rosrun image_pkg hsv_node_beta.py
```

### 3️⃣ Перемещение целевого шара (опционально) ⚽

```bash
rosrun image_pkg move_ball.py _model_name:=green_ball
```

### 4️⃣ Запись траектории робота 📈

```bash
rosrun image_pkg plot_trajectory.py \
    _output_path:=~/trajectory.png \
    _target_model_name:=green_ball
```

### 5️⃣ Оценка производительности 📊

```bash
rosrun image_pkg tracking_metrics.py \
    _target_color_name:=green \
    _auto_stop_on_reached:=true
```



## 🎮 Инструкция по использованию

После запуска `hsv_node_release.py` появятся три окна OpenCV:

| Окно | Функция |
|:---|:---|
| 🎚️ **Threshold** | Выбор целевого цвета с помощью ползунка (`0`=вручную, `1`=💚зелёный, `2`=🧡оранжевый, `3`=❤️красный, `4`=💙синий) |
| 🖼️ **RGB** | Отображение результатов обнаружения (зелёная рамка ✅, синий крест в центре 🔵, информация о состоянии) |
| 📊 **Result** | Бинарное изображение после пороговой сегментации HSV |

Робот автоматически переходит в цикл состояний:

```
🔍 SEARCHING ──► 🎯 ALIGNING ──► 🏃 APPROACHING ──► ✅ REACHED
```
